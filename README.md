# Currency_Converter
# 💱 Currency Converter

A modern and responsive Currency Converter web application built with **HTML, CSS, and JavaScript**. It uses the **Frankfurter API** to fetch real-time exchange rates and provides a clean, user-friendly interface for quick currency conversions.

## 🚀 Features

- 🌍 Real-time currency conversion
- 🔄 One-click currency swap
- 🚩 Country flags for each currency
- ⏳ Loading animation while fetching exchange rates
- 📋 Copy converted value
- 📱 Fully responsive design
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Frankfurter Exchange Rate API

## ⚡ Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/currency-converter.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup is required.

## 🌐 API Used

This project uses the **Frankfurter API** to fetch live exchange rates.

https://frankfurter.dev/
